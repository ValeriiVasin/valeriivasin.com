# valeriivasin.com

## Quickstart
1) Use the project's Node version:
   - `nvm install`
   - `nvm use`
2) Enable Corepack (one-time per machine):
   - `corepack enable`
3) Use the pinned pnpm version:
   - `corepack prepare pnpm@10.28.1 --activate`
4) Install dependencies and start dev server:
   - `pnpm install`
   - `pnpm dev`

Open the dev server URL shown in the terminal.
