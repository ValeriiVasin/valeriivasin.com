# valeriivasin.com

## Quickstart
1) Use the project's Node version:
   - `nvm install`
   - `nvm use`
2) Install the pinned pnpm version:
   - `npm install --global pnpm@10.28.1`
3) Install dependencies and start dev server:
   - `pnpm install`
   - `pnpm dev`
4) Format or validate formatting:
   - `pnpm format`
   - `pnpm check`

Open the dev server URL shown in the terminal.
